# 📋 知识整理技能

**技能名称**: knowledge-organizer  
**版本**: 1.0  
**创建时间**: 2026-04-19  
**作者**: 小蟹 🦀

---

## 📋 技能描述

使用 LLM 理解和组织公司知识库信息，提供结构化整理报告。

**核心能力**:
- 智能文档定位（关键词组匹配）
- 上下文信息提取（保留前后文）
- LLM 语义理解（不是规则匹配）
- 结构化报告生成
- 信息缺口识别

---

## 🎯 触发条件

当用户提问包含以下模式时触发：

**查询类**:
- "X 是什么？"
- "介绍一下 X"
- "X 的详情/信息/数据"

**整理类**:
- "整理一下 X"
- "总结 X"
- "X 的情况/进展/进度"

**分析类**:
- "分析 X"
- "X 怎么样"
- "X 有哪些"

## 💡 技能特点

1. **LLM 驱动** - 用语义理解而非规则匹配
2. **结构化输出** - 清晰的报告格式
3. **缺口识别** - 明确指出缺失信息
4. **来源标注** - 每个信息点标注来源
