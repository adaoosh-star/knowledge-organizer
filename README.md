## 🎯 功能说明

使用 LLM 理解和组织公司知识库信息，提供结构化整理报告。

**核心能力**:
- ✅ 智能文档定位（关键词组匹配）
- ✅ 上下文信息提取（保留前后文）
- ✅ LLM 语义理解（不是规则匹配）
- ✅ 结构化报告生成
- ✅ 信息缺口识别

## 🎯 技术实现

### 工作流程

```
用户提问
  ↓
1. 生成智能关键词
  ↓
2. 扫描相关文档（精准匹配）
  ↓
3. 提取上下文（前后各 10 行）
  ↓
4. 构建 LLM Prompt
  ↓
5. LLM 理解并组织信息
  ↓
6. 生成结构化报告
```

### 核心算法

**1. 智能关键词匹配**
```python
# 不是简单匹配，而是关键词组
keywords = ['518 周年庆', '5·18', '品牌焕新', 'MSH 专场']
match_count = sum(1 for kw in keywords if kw in content)
```

**2. 上下文提取**
```python
# 提取关键词前后各 10 行，保留完整信息
start = max(0, i - 10)
end = min(len(lines), i + 11)
context = '\n'.join(lines[start:end])
```

**3. LLM 语义理解**
```python
# 用 LLM 理解并组织信息，不是规则匹配
prompt = f"""请理解并整理以下关于"{topic}"的信息..."""
```

## 🔒 安全架构

本技能采用**配置注入**架构，确保代码通用性和安全性：

```
skill.py (通用代码)
    ↓
读取 config.json
    ↓
获取路径配置
    ↓
访问知识库
```

**安全说明**:
- ✅ `skill.py` 是通用代码，不包含公司特定信息
- ✅ 公司知识路径通过 `config.json` 配置注入
- ✅ `config.json` 应加入 `.gitignore`，不提交到 Git
- ✅ 分享技能时，只提供 `config.json.example` 模板

详见：[SECURITY.md](SECURITY.md)
